# COVID-19 Data Analysis

## Project Overview

This project analyzes a COVID-19 dataset using Python to explore country-wise pandemic statistics. It includes data loading, cleaning, exploratory data analysis (EDA), and data visualization to identify trends and gain insights from the dataset.

## Features

* Load and analyze CSV data using Pandas
* Explore and summarize COVID-19 statistics
* Perform Exploratory Data Analysis (EDA)
* Create visualizations using Matplotlib and Seaborn
* Identify trends and patterns across countries

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Dataset

**File:** `country_wise_latest.csv`

The dataset contains country-wise COVID-19 statistics, including confirmed cases, deaths, recovered cases, active cases, and other related information.

## How to Run

1. Clone the repository.
2. Open `project.ipynb` in Google Colab or Jupyter Notebook.
3. Install the required libraries:

```bash
pip install pandas matplotlib seaborn
```

4. Run all notebook cells.

## Results

This project provides visual insights into COVID-19 statistics using tables,charts and graphs, helping to understand trends and compare country-wise data.
